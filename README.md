# Wordle
Clone du jeu Wordle

### Setup
1. In root project, create virtual environment 
``` Python
python3 -m venv <env_name>
```
Or
``` Python
virtualenv <env_name>
```
2. Activate environment 
``` Python
source <env_name>/bin/activate
```
3. Install the dependencies
``` Python
pip install -r requirements.txt
```

### Run the game

``` Python
python3 src/main.py
```
