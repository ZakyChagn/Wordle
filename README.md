# Wordle
Clone du jeu Wordle

### Setup
1. In root project, create virtual environment 
```
python3 -m venv <env_name>
```
Or
```
virtualenv <env_name>
```
2. Activate environment 

Linux
``` 
source <env_name>/bin/activate
```
Windows
```
.\<env_name>\Scripts\activate
```

3. Install the dependencies
```
pip install -r requirements.txt
```

### Run the game

```
python src/main.py
```
